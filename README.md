# 청라 라이프 가이드 미션

모바일 기준 정적 웹페이지입니다.

## 수정 내용

- 하나금융그룹 청라 HQ를 `본부 공항`으로 표현
- `런치 Gate`, `미팅 Gate`, `애프터 Gate`, `하나로컬 Gate` 추천 장소 카드 구성
- 각 장소 카드에 `차로/도보 이동시간 + 거리`가 바로 보이도록 표시
- 본부 내부 장소는 Coming Soon 상태로 유지
- Gate 선택 룰렛 기능 포함
- 네이버지도 검색 링크 연결
- 모바일에서 글자가 겹치거나 흰 배경에 흰 글자가 들어가지 않도록 대비 개선

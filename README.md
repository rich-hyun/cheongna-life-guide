## 실행
프로젝트 폴더에서 아래 명령을 실행합니다.

```bash
py -3 -m http.server 8007
```

브라우저에서 `http://localhost:8007/`로 접속하세요.
GitHub Pages에는 이 폴더 안의 파일과 `assets` 폴더를 구조 그대로 올려야 합니다.
